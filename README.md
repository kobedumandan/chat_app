A simple chat application made using Flutter and Firebase.

To install:
  * Either run the code through VSCode or install the .apk file

To install through VSCode:
  Pre-requisites:
   * Must have Flutter SDK installed
   * Must have Android studio installed
   * Must have Git installed
     
   * Download the files by pressing Code -> Download ZIP.
   * Extract the files then open the extracted folder through VSCode.
   * Choose a device to use (Emulator or External device).
   * Run "flutter run" (if you have installed the Flutter Extension, open main.dart then press the Start Debugging button).

Or if you want to install directly:
 * Open this link: https://drive.google.com/drive/folders/1D2E1igce1i_C7aqkk24CvckekbjWCnPQ?usp=sharing
 * Download chat_app.apk and install on your phone.
